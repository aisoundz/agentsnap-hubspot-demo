Ready to try it with YOUR HubSpot? Install in 5 minutes.

# 1. Download & unzip
curl -L -o agentsnap-hubspot-demo.zip \
  https://github.com/aisoundz/agentsnap-hubspot-demo/releases/download/v1.0/agentsnap-hubspot-demo.zip
unzip agentsnap-hubspot-demo.zip
cd agentsnap-hubspot-demo

# 2. Python environment
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# 3. Verify
python -c "import server; print('OK')"

# 4. Wire into Claude Desktop — see INSTALL.md
#    Restart Claude. Ask: "What's on my plate today?"
