Install in 5 minutes

You need Python 3.10+ and Claude Desktop. Mock data works out of the box; live HubSpot mode takes an extra 2 minutes for the private app token.

# 1. Unzip wherever you like
cd ~/Documents/agentsnap-hubspot-demo

# 2. Python environment
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 3. Sanity check
python -c "import server; print('OK')"

# 4. Wire into Claude Desktop — see INSTALL.md
#    Restart Claude. Done.

Why MCP, not another web app

Most insurance AI tools today are standalone platforms — you log into one more browser tab and copy-paste between it and your AMS or CRM. That's friction your team won't sustain.

AgentSnap is built on the Model Context Protocol, which means it runs inside the chat interface your team already uses — Claude Desktop today, ChatGPT and any other MCP-capable client tomorrow. Same prompt window, same workflow. The integration disappears.

And because the tools call out to HubSpot and SnapRefund's payment infrastructure under the hood, the AI is doing real work — creating real deals, writing real timeline notes, advancing real renewals — not generating prose about what it would do.
